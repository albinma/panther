## Getting Started

Copy env file:

```bash
cp .env .env.local
```

Run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Testing

If running test for the first time:

```bash
npx playwright install
```

To run tests:

```bash
yarn test:e2e
```
